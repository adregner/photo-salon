# Roadmap

Planned features not yet implemented in photo-salon.

## Upcoming

- **Auto-rotate** - rotate the image as specified in file metadata.  Keyboard shortcuts to rotate 90 degrees in either direction, or to flip horizontally or vertically.

## Done

- **EXIF metadata** - Press `E` to toggle a semi-transparent overlay showing EXIF metadata (camera, exposure, focal length, date, etc.) plus basic file info.
