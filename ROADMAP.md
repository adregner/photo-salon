# Roadmap

Planned features not yet implemented in photo-salon.

## Upcoming

- **Expanded format support** — PNG, TIFF, WebP, and other formats beyond JPG
- **Folder navigation** — open the next or previous image in the same folder as the current file
- **Open folder** — open a folder path instead of a single image file, start displaying the first image in the folder and press a keyboard shortcut to display a menu to select another file in the folder
- **Fullscreen** — press keyboard shortcut "F" to enter fullscreen (no titlebar, no statusbar, no start menu)
- **Adjustable background color** — start with solid black background, press keyboard shortcut to show controls to select greyscale value
- **Crop tool** — rubber-band selection to define a crop region; save cropped output to a new file
