# Crop Tool Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Press **C** to enter crop mode; rubber-band drag to define a region; press **Enter** to save the cropped image to a user-chosen file; press **Escape** to cancel.

**Architecture:** `ImageViewer` manages crop state internally (`m_cropMode`, `QRubberBand`, drag origin). It overrides `mousePressEvent`, `mouseMoveEvent`, and `mouseReleaseEvent` to build the rubber band in viewport coordinates. On **Enter** it emits `cropConfirmed(QRectF sceneRect)` after mapping viewport → scene coordinates. `MainWindow` receives the signal, retrieves the current `pixmap()` from `ImageViewer`, clips the crop rect, and prompts for a save path via `QFileDialog`.

**Tech Stack:** Qt6 (`QRubberBand`, `QGraphicsView::mapToScene`, `QFileDialog`, `QPixmap::copy`), QTest, C++17, CMake

```cpp
void ImageViewer::setCropMode(bool active) {
    if (m_cropMode == active) return;
    m_cropMode = active;
    if (active) {
        setDragMode(QGraphicsView::NoDrag);
        viewport()->setCursor(Qt::CrossCursor);
    } else {
        if (m_rubberBand) {
            m_rubberBand->hide();
        }
        setDragMode(QGraphicsView::ScrollHandDrag);
        viewport()->unsetCursor();
    }
    emit cropModeChanged(m_cropMode);
}

bool ImageViewer::rubberBandVisible() const {
    return m_rubberBand && m_rubberBand->isVisible();
}

void ImageViewer::mousePressEvent(QMouseEvent *event) {
    if (!m_cropMode) { QGraphicsView::mousePressEvent(event); return; }
    if (event->button() == Qt::LeftButton) {
        m_cropOrigin = event->pos();
        if (!m_rubberBand)
            m_rubberBand = new QRubberBand(QRubberBand::Rectangle, viewport());
        m_rubberBand->setGeometry(QRect(m_cropOrigin, QSize()));
        m_rubberBand->show();
        event->accept();
    }
}

void ImageViewer::mouseMoveEvent(QMouseEvent *event) {
    if (!m_cropMode || !m_rubberBand) { QGraphicsView::mouseMoveEvent(event); return; }
    m_rubberBand->setGeometry(QRect(m_cropOrigin, event->pos()).normalized());
    event->accept();
}

void ImageViewer::mouseReleaseEvent(QMouseEvent *event) {
    if (!m_cropMode) { QGraphicsView::mouseReleaseEvent(event); return; }
    event->accept();
    // Rubber band stays visible — user confirms with Enter
}
```

```cpp
connect(viewer, &ImageViewer::cropConfirmed, this, [this, viewer](const QRectF &sceneRect) {
    QPixmap original = viewer->pixmap();
    if (original.isNull()) return;

    // Clamp crop rect to actual image bounds
    QRect cropRect = sceneRect.toAlignedRect().intersected(original.rect());
    if (cropRect.isEmpty()) {
        QMessageBox::warning(this, "Crop", "Selected region is outside the image.");
        return;
    }

    QPixmap cropped = original.copy(cropRect);

    QString savePath = QFileDialog::getSaveFileName(
        this,
        QStringLiteral("Save Cropped Image"),
        QFileInfo(viewer->currentPath()).dir().absoluteFilePath(
            QFileInfo(viewer->currentPath()).baseName() + "-cropped.jpg"),
        QStringLiteral("JPEG Images (*.jpg *.jpeg);;PNG Images (*.png);;All Files (*)"));

    if (savePath.isEmpty()) return;

    if (!cropped.save(savePath)) {
        QMessageBox::critical(this, "Crop", QString("Failed to save: %1").arg(savePath));
    }
});
```

## Coordinate Mapping Note

`mapToScene(QRect viewportRect)` returns a `QPolygonF`. Calling `.boundingRect()` on it gives back the corresponding `QRectF` in scene (image) coordinates, correctly accounting for any zoom level and scroll position. The image pixmap is placed at the scene origin, so scene coordinates equal image pixel coordinates directly — no further offset is needed.
